## Metric spaces:
We want to give the set $X$ more structure, in this case we want to know what the `distance` between two chosen element is.

> Let X is a set. A metric: $d: X \times X \mapsto [0, \infty )$  s.t with $x, y, z \in X$
> 1. $d(x, y) = 0 \Leftrightarrow x = y$
> 2. $d(x, y) = d(y, x)$
> 3. $d(x, y) \le d(x, z) + d(z, y)$
